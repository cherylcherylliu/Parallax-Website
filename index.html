視差滾動效果

作者：Vineet Kumar

什麼是視差滾動？

視差滾動是一種網頁設計技巧，網站背景的移動速度比前景更慢，讓訪客向下捲動時能呈現 3D 效果，增添深度並打造更具沉浸感的 瀏覽體驗。視差是基於視覺錯覺：人眼會覺得靠近的物體比遠處物體大，因此也會覺得遠處的物體移動得更慢。這種錯覺早已被應用於不同媒 介的視差效果，營造寫實感。最早可追溯到迪士尼《白雪公主與七矮人》這類傳統動畫，之後也常見於《超級瑪利歐》等電玩遊戲。隨著 CSS 與 HTML 的發展，視差效果進一步演化，成為我們今日熟悉的網頁設計手法。

最早在瀏覽器實作視差滾動的是開發者 Glutnix 在 2007 年於部落格分享的範例程式碼與示範，使用 JavaScript 與 CSS 2，並支援當 時的 Internet Explorer 6 及其他瀏覽器。不過直到 2011 年 HTML5 與 CSS3 推出後，這項技術才在網頁設計領域普及。支持者認為這是擁抱網 路流動性的一種簡單方式，並把視差背景當成吸引用戶、提升網站體驗的工具。然而，普渡大學 2013 年的研究指出：「……儘管視差滾動強化了 用戶體驗的某些面向，但不一定能提升整體用戶體驗。」


為什麼要使用視差圖形？

    設計師應該嘗試視差圖形的兩個主要原因：

  • 有助於講故事

    視差圖形非常適合用來說故事。好的設計就是好的溝通——在設計網站時，我們希望與訪客交流，講述關於產品、服務或品牌的 動人故事。透過視差滾動，我們能讓訪客沉浸在獨特的線上體驗中，甚至可以打造單頁網站，讓讀者不必切換頁面就能閱讀品牌或產 品的完整故事。

  • 有助於提升使用者參與度

    視差圖形也能提升使用者參與度。降低網站的跳出率是網頁設計需要留意的目標之一。跳出率衡量的是訪客只看一頁就按下「上一頁」 或關閉分頁的比例。高跳出率會影響搜尋排名，因此任何能降低跳出率的做法都值得嘗試。設計精良的視差頁面能帶來更具動態的體驗， 讓使用者停留更久。


如果你想在網站加入視差滾動效果，應考慮以下提示。

  • 選擇性使用

    將視差滾動運用在特定元素上，比起所有頁面、任何時刻都使用更有威力。可用於頁首、標題或僅限首頁，目標是 強化網站設計，而不是分散使用者注意力。

  • 圖像壓縮

    視差滾動會使用多種媒體檔案與 CSS 形狀來營造深度感。避免使用過大的圖片檔，以免拖慢載入速度，犧牲效能與 使用者體驗。

  • 善用色彩

    視差滾動不僅仰賴前景與背景的圖像，也涉及色彩運用。背景可以使用較低彩度的色調，前景則採用更鮮活的顏色， 進一步強化深度效果。
